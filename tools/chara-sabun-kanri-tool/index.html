キャラ差分管理ツール

立ち絵・表情差分をまとめて確認し、ファイル名を整えて、CCFOLIA向けの @差分 チャットパレットを生成します。

←TRPG WEBツール観測所 ×
v0.032 / 2026.06.11

使い方

  1. 画像をドラッグ&ドロップ、またはクリックして複数画像を選択します。
  2. メインファイル名を入力し、各サムネイル横の差分名を編集します。
  3. 必要に応じて「番号追加」をONにすると、メインファイル名01_差分名.png の形式になります。
  4. 「リネーム画像をZIPで出力」で画像一式を保存できます。
  5. 「@差分チャパレをコピー」で、CCFOLIAなどに貼り付ける差分切替用チャットパレットをコピーできます。
