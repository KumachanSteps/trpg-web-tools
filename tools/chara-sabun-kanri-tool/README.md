# キャラ差分管理ツール v0.13

TRPG立ち絵・表情差分の画像確認、ファイル名整理、CCFOLIA向け @差分 チャットパレット作成を支援するWebツールです。

## v0.13 / 2026.06.11

- メイン3パネルの表示サイズを固定し、差分を多数追加してもパネル自体が伸びないよう調整
- サムネイル一覧はパネル内スクロールで表示
- 差分名サジェストは大プレビュー下部の表示領域を広く使い、必要時はスクロールで表示
- 差分名サジェストを以下に更新：通常、笑顔、怒り、泣き、驚き、困惑、照れ、焦り、微笑、真剣、悲哀、喜び、呆れ、不安、目閉じ、ジト目、負傷、戦闘、発狂、絶望

## 使い方

1. `index.html` をブラウザで開きます。
2. 画像をドラッグ&ドロップ、またはクリックして複数画像を選択します。
3. メインファイル名と差分名を入力します。
4. 必要に応じて「番号追加」をON/OFFします。
5. ZIP出力、または @差分チャパレコピーを実行します。
