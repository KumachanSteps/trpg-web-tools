- フッターパネルを追加

## 使い方

1. `index.html` をブラウザで開く。
2. 画像をドラッグ&ドロップ、またはクリックして複数選択。
3. メインファイル名と差分名を編集。
4. 必要に応じて番号追加をON。
5. ZIP出力、または `@差分` チャットパレットをコピー。

※ ZIP生成には CDN版 JSZip を利用しています。
