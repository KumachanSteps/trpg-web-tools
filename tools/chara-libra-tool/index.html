キャラクター情報を保管・編集し、CCFOLIA駒データとして再出力します。

v1.0 / 2026.06.19

使い方

  1. 左のControl DockにCCFOLIA駒JSONを貼り付けます。
  2. 「駒データを登録 / 更新」を押すと右側のCharacter Display Fieldに登録されます。
  3. 検索・フィルター・ソート、アイコンビュー / カードビュー切替でキャラクターを探せます。
  4. キャラクターをクリックすると詳細編集ページへ移動します。

キャラクターデータはお使いのブラウザ内に保存されます。必要に応じて詳細ページなどからJSONバックアップを保存してください。
