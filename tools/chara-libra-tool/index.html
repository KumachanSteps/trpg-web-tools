キャラクター情報を保管・編集し、CCFOLIA駒データとして再出力します。

v1.0 / 2026.06.19

使い方

  1. CCFOLIA駒JSONを貼り付けて登録します。
  2. カードをクリックして詳細を開きます。
  3. 基本情報、MEMO、チャットパレットを編集して保存します。
  4. CCFOLIA駒データコピーで更新済みJSONをコピーします。

キャラクターデータはお使いのブラウザ内に保存されます。必要に応じてJSONバックアップを保存してください。

ショートカット

Esc パネルを閉じる / 一覧へ戻る

Ctrl/Cmd + S 詳細を保存

Ctrl/Cmd + Shift + T テーマ切替
