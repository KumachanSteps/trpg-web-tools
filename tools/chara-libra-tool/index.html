←TRPG WEBツール観測所
v1.61 / 2026.06.20

使い方

  1. 左カラムにCCFOLIAのキャラクター駒データJSONを貼り付けます。
  2. 「駒データを登録 / 更新」を押すと、キャラクターがライブラリーに追加されます。
  3. 右側のフィールドで検索・ソート・表示切替ができます。
  4. キャラカードをクリックすると、MEMO / チャパレ編集とCCFOLIA駒データコピーができます。

データはブラウザ内に保存されます。必要に応じてJSONバックアップを保存してください。
