✦A卓報告ジェネレータ

SNSの卓報告パターンをもとに、複数スタイルのX向け報告文を生成できます。 シナリオ名、参加者、END、装飾スタイルを入力して、コピーしやすい卓報告文を作成します。

入力

SNS実例の構造だけを参考にしたテンプレートです。

英数字のみ変換します。日本語・記号・装飾はそのままです。

敬称

緑の＋で進行役を追加できます。KPC/KP型では、名前欄に「KPC名 / KP名」のように入れると綺麗です。

HO、PC名、PL名を1行で入力します。HO補足は通常未入力でOKです。

選択肢は PC / PC1 / HO1 / PC/PL / PL/PC / 自由です。1行目が PC1 なら PC2, PC3、HO1 なら HO2, HO3 のように自動で続きます。

クリックすると、右側プレビューの編集中カーソル位置に挿入される想定です。
