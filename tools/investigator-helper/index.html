探索者ヘルパーv1.3

職業サンプル、特徴表、経験パッケージを検索・閲覧し、キャラシ貼り付け用に整形できるクローズド環境向け支援ツール。

使い方

対象ルールを選び、キーワード・出典・技能チップで職業サンプルを検索します。選択した情報は右側のメモ欄へ追加し、編集してからコピーできます。

ショートカット

Esc：パネルを閉じる / Ctrl または Cmd + Enter：出力欄をコピー

対象ルール
技能チップ検索（全て）

特徴表ランダム抽出

1d6 + 1d10で2〜5個を抽出します。重複は自動的に避けます。

経験パッケージ

7版探索者用の経験パッケージを閲覧し、出力欄へ追加します。

CoC 7版用データ

キャラシ貼り付け用メモ
