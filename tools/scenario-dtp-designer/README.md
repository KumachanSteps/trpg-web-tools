# シナリオDTPデザイナー v1.6

TRPGシナリオ本文をA4縦・本文2カラム・右KPメモ欄のDTP風レイアウトで確認、編集する静的HTML版プロトタイプです。

## v1.6 更新内容

- v1.4の反映不足を確認し、保存キーを更新して旧localStorageの影響を受けにくくしました。
- デフォルトサンプルを `雨待ち停留所` の架空シナリオ本文に変更しました。
- 初期表示では1ページ目のみ編集可能なヘッダーブロックを配置します。
- 「プレビューに読み込む」では本文をブロック化せず、TXT内容に沿った2カラム流し込み表示にします。
- ブロック化は「選択中の表示文をブロック化」ボタンで編集時のみ実行する運用に整理しました。
- 左カラム①のTXT編集欄と下部ボタンの余白を削減し、パネルとフッターがつながるように再調整しました。
- ページ間隔をさらに狭めました。
- 初期表示ズームを画面の高さに合わせて調整するようにしました。

## v1.6 追加修正

- ヘッダー追加時の初期文言を `雨待ち停留所` に統一しました。
- 旧サンプル `イザナミの棺` が残らないよう保存キーを更新しました。
- プレビュー上の流し込み本文を直接編集可能にしました。
- プレビュー上のブロックをクリックした際、右カラムの編集欄へ自動フォーカスします。
- KPメモのデフォルト文字サイズを8pxにしました。
- ヘッダーや見出しブロックを追加しても流し込み本文がリセットされないよう修正しました。
- `選択中の表示文をブロック化` は、プレビュー上で選択した範囲をブロック化します。
- クリア後もTXTファイル読込エリアが再表示されるよう調整しました。
