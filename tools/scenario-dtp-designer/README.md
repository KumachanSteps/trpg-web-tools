# シナリオDTPデザイナー v1.4

TRPGシナリオ本文を、A4縦・本文2カラム・右KPメモ欄のDTP風レイアウトへ配置する静的HTML版です。

## v1.4 更新内容

- ヘッダーを通常ブロック化し、初期状態では1ページ目のみに表示
- ヘッダーブロックを「ブロック追加」から追加可能
- TXTファイル読込後、ドラッグ＆ドロップ欄を非表示化
- テキスト編集欄を左カラム①全体に広げる構成へ調整
- ページ一覧パネルを5ページ程度の高さに固定し、パネル内スクロール化
- 左カラム①/②をドラッグで幅変更、ダブルクリックで初期幅に復帰
- UIフォントサイズとボタンサイズを全体的に小さく調整
- プレビュー初期ズームを縮小
- ルーラー幅をコンパクト化
- KPメモをポストイット風から上下ラインのみのシンプルデザインへ変更
- KPメモブロックもドラッグで縦位置調整可能
- 「プレビューに読み込む」の自動配置ロジックを改善
- 勝手なページごとのヘッダー生成を停止
- ブロックドラッグ時にグリッドスナップとページ内制約を適用

## v1.4 更新内容

- TXT読込後の左カラム①をテキスト編集欄中心の構成へ再調整
- [プレビューに読み込む] [クリア] を下辺に固定し、同じ高さで表示
- 本文2カラムの中央に薄い縦ガイド線を追加
- ブロック移動時、中央ガイドを基準に左/右カラムへスナップ
- プレビュー読み込み時の生成順とブロック高さ推定を改善
- 生成ブロックのテキスト量に合わせて高さを自動調整
- プレビュー上の選択ブロックを直接編集可能化
- プレビュー上の編集内容と右インスペクターのテキスト欄をリアルタイム同期
- 右インスペクターのテキスト欄を内容に合わせて伸縮、最大高さを設定


## v1.4 updates
- プレビュー読み込み時はブロック化せず、TXT本文を2カラムの流し込み表示として出力します。
- 左カラム①のTXT編集欄と下部ボタンの余白を削減しました。
- ページ間隔を狭め、初期ズームをA4縦プレビューが見やすい値に調整しました。
- 「表示本文をブロック化」ボタンで、編集時のみブロック化できます。
