# シナリオDTPデザイナー v1.2

TRPGシナリオ本文を、A4縦・本文2カラム・右KPメモ欄のDTP風レイアウトへ配置する静的HTML版です。

## v1.2 更新内容

- ヘッダーを通常ブロック化し、初期状態では1ページ目のみに表示
- ヘッダーブロックを「ブロック追加」から追加可能
- TXTファイル読込後、ドラッグ＆ドロップ欄を非表示化
- テキスト編集欄を左カラム①全体に広げる構成へ調整
- ページ一覧パネルを5ページ程度の高さに固定し、パネル内スクロール化
- 左カラム①/②をドラッグで幅変更、ダブルクリックで初期幅に復帰
- UIフォントサイズとボタンサイズを全体的に小さく調整
- プレビュー初期ズームを縮小
- ルーラー幅をコンパクト化
- KPメモをポストイット風から上下ラインのみのシンプルデザインへ変更
- KPメモブロックもドラッグで縦位置調整可能
- 「プレビューに読み込む」の自動配置ロジックを改善
- 勝手なページごとのヘッダー生成を停止
- ブロックドラッグ時にグリッドスナップとページ内制約を適用
