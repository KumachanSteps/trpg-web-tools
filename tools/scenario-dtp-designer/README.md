# シナリオDTPデザイナー v1.5

TRPGシナリオ本文をA4縦・本文2カラム・右KPメモ欄のDTP風レイアウトで確認、編集する静的HTML版プロトタイプです。

## v1.5 更新内容

- v1.4の反映不足を確認し、保存キーを更新して旧localStorageの影響を受けにくくしました。
- デフォルトサンプルを `雨待ち停留所` の架空シナリオ本文に変更しました。
- 初期表示では1ページ目のみ編集可能なヘッダーブロックを配置します。
- 「プレビューに読み込む」では本文をブロック化せず、TXT内容に沿った2カラム流し込み表示にします。
- ブロック化は「表示本文をブロック化」ボタンで編集時のみ実行する運用に整理しました。
- 左カラム①のTXT編集欄と下部ボタンの余白を削減し、パネルとフッターがつながるように再調整しました。
- ページ間隔をさらに狭めました。
- 初期表示ズームを画面の高さに合わせて調整するようにしました。
