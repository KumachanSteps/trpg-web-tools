シナリオDTPデザイナー v1.0 static prototype

テキストを読み込み、2カラム本文＋右KPメモ欄のDTPページへ配置する試作版です。

使い方： 左の「テキスト取込」にシナリオ本文を貼り付ける → 自動ブロック化で構造を生成 → 中央のページでレイアウトを調整 → 右の設定で見た目を調整 → 出力（PDF/PNG）できます。
ショートカット：Ctrl/Cmd+S 保存、Ctrl/Cmd+O JSON読込、Ctrl/Cmd+Shift+P PDF出力、Ctrl/Cmd+Shift+I 画像追加、Esc 選択解除/パネル閉じ。
