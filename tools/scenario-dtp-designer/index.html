シナリオDTPデザイナー v1.6

A4縦・本文2カラム・右KPメモ欄で、シナリオ本文をDTP風に整える静的HTML版です。

使い方： 左カラム①でTXTを開く/ドラッグ＆ドロップする → テキストを確認・編集 → プレビューに読み込む → 中央のA4縦プレビューで配置調整 → 右の設定で見た目を調整 → PDF/PNG/JSONで保存できます。 プレビューは空白部分をドラッグして移動、Cmd/Ctrl + スクロールで拡大縮小できます。
Cmd/Ctrl + SJSON保存 Cmd/Ctrl + OJSON読込 Cmd/Ctrl + Z元に戻す Cmd/Ctrl + Shift + Z / Ctrl + Yやり直し Delete / Backspace選択ブロック削除 Cmd/Ctrl + Shift + PPDF出力 Cmd/Ctrl + Shift + EPNG出力 Cmd/Ctrl + Wheelプレビュー拡大/縮小 Esc選択解除 / パネルを閉じる
