卓ログトラッカー

←TRPG WEBツール観測所 𝕏

遊んだ卓を記録し、感想・卓報告・プレイ済みリストへつなげるログ管理ツール。テーブルで履歴を整理し、各行から「卓報告ジェネレーター」へ連携できます。

v1.61 2026.06.21

HOW TO USE

使い方

1卓ログテーブルで遊んだ卓の情報を追加・編集します。
2＋ 項目追加 / − 項目削除で表示する列を調整できます。
3右端の送信ボタンから卓報告ジェネレーターへ連携できます。

SHORTCUTS

ショートカット

Escドロワー / ショートカットパネルを閉じる
Ctrl/⌘ + Shift + Sショートカットパネルを開く
Ctrl/⌘ + Shift + N新しいセッションログを追加
Ctrl/⌘ + Shift + F検索欄へ移動
Ctrl/⌘ + JJSON入力
Ctrl/⌘ + EJSON出力

卓ログテーブル

「＋ 項目追加」からお気に入り、HO、エンディング、リンク、カスタム項目などを追加できます。

ドラッグ操作：項目ヘッダーの ⋮⋮ を掴んで、表示したい位置にドロップできます。
追加データ項目

テーブルに追加する任意項目を選択できます。これらの項目は卓報告ジェネレーターにも渡せます。

表示項目を削除

チェックを外すとテーブル列だけ非表示になります。入力データは保持されます。

詳細メモ・感想

選択中の卓の詳細

プレイ済みシナリオ一覧出力

プレイ済みシナリオ一覧を、日付順・システム別・GM別・PL別などでテキスト出力できます。

ブラウザ保存について

このツールのデータは、お使いのブラウザ内に保存されます。別端末・別ブラウザには自動同期されません。大切な記録は、定期的にJSON出力でバックアップしてください。
