使い方

基本の流れ

  1. 左欄に、シナリオ本文やPDFからコピーしたNPCデータを貼り付けます。
  2. 解析する を押すと、中央欄にNPC名・ステータス・戦闘・技能が整理されます。
  3. 中央欄の内容を編集すると、右欄のチャットパレットとNPC駒データへ自動反映されます。
  4. チャットパレット欄は直接編集できます。編集内容はJSONの commands に反映されます。
  5. コピー または NPC駒データコピー で、CCFOLIA等へ貼り付けます。

安全仕様

CCFOLIA駒の memo は、秘匿情報の漏洩を防ぐため、初期値をNPC名のみにしています。

装甲や技能など、内容が空のカテゴリはチャットパレットに出力されません。

対応している表記

近接戦闘（格闘）60%（30/12）、ダメージ 1D3+DB のような公式シナリオ風の表記に対応します。括弧内のハード/イクストリーム値は自動で除外します。

1. NPC情報を貼り付け
PDF / TXT対応

PDF由来の改行崩れ、全角スペース、括弧内のハード/イクストリーム値をある程度吸収します。

使い方
  1. シナリオ本文やNPCデータを左欄に貼り付けます。
  2. 解析後、中央欄で名前・ステータス・技能・メモを確認します。
  3. 中央欄の編集は右欄のチャットパレットとJSONに反映されます。
  4. チャットパレット欄は直接編集できます。
  5. JSONは直接編集不可。編集済みチャパレはJSONの commands に反映されます。
  6. CCFOLIA駒の memo は秘匿漏れ防止のため、初期値をNPC名のみにします。
