KP用キャラシビューアー

いあきゃらのコマ出力データを貼り付けると、KP用のPCカードをすぐ表示します。

いあきゃら コマ出力データを貼り付け

このエリア上で Ctrl / Cmd + V すると、いあきゃらのコマ出力データを読み込んでカードを生成します。カードはこのレイヤーの上に表示されます。

すべてのPCカードを削除しますか？

現在表示されているPCカードをすべて削除します。この操作は取り消せません。

手動コピーが必要です

この環境ではClipboard APIがブロックされています。下のテキストを選択済みなので、Ctrl / Cmd + Cでコピーしてください。
