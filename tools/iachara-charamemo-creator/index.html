使い方

  1. いあきゃらの「キャラ出力▶ココフォリア駒出力」から駒JSONをコピーし、左上へ貼り付けます。
  2. いあきゃらのテキスト出力を左下へ貼り付けるか、txtファイルを開きます。
  3. 生成オプションで含めたい項目を選び、キャラメモ編集・プレビューを確認します。
  4. 必要ならキャラメモやチャットパレットを直接編集し、最後に生成駒JSONデータをコピーします。

Ctrl / Cmd + Shift + D

Esc
