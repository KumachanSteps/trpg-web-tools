使い方

  1. 「キャラ出力▶︎ココフォリア駒出力」からコピーした駒JSONを貼り付けます。
  2. いあきゃらのテキスト出力を貼り付けるか、txtファイルを開きます。
  3. 必要に応じて生成オプションを切り替え、キャラメモを編集します。
  4. 「生成駒JSONデータをコピー」からCCFOLIAへ貼り付けます。

ショートカット

Esc

Ctrl / Cmd + Shift + C

Ctrl / Cmd + Shift + R
入力データを削除
Ctrl / Cmd + Shift + T
ライト/ナイトモードを切り替え
