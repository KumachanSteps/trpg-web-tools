CoC6版/7版成長ツール

セッションログから、探索者ごとの「成長チェック候補技能」を抽出します。CoC6版・7版のハウスルール運用に合わせて、成功・クリティカル・ファンブルの表示条件を切り替えできます。

Shortcut: Ctrl / Cmd + O でファイル選択、Ctrl / Cmd + Enter で解析、Ctrl / Cmd + Shift + M でナイトモード切替。

入力欄

成長チェック候補の表示ルール

除外ルール

キャラクター表示

この回数未満のキャラクターは初期状態で非表示になります。

ログを入力して「成長チェックを抽出」を押してください。
