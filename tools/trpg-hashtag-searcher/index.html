使えるハッシュタグ検索

TRPGでよく使うX検索をワンクリックで作成できます。ハッシュタグ、検索ワード、フィルター、ネタバレ回避用の除外ワードを組み合わせて、目的に合った検索をすばやく開けます。

ショートカット

キーボード操作で検索・コピー・パネル表示を素早く行えます。

⌘ / Ctrl + Kショートカットパネルを開閉
⌘ / Ctrl + Enter現在の検索文でX検索を開く
⌘ / Ctrl + Shift + C現在の検索文をコピー
Escショートカットパネルを閉じる
